# 1-1 Meeting Scheduler

Schedule 30-minute 1-1 meetings between a visitor and department colleagues.

## To sign up

1. Open https://remy.wang/sched
2. Enter your name and select available time slots (drag to select multiple)
3. Click **Download Availability** and email the `.json` file to the organizer

## For the organizer

1. Open https://remy.wang/sched/mksched.html
2. Mark the visitor's available time slots
3. Upload all colleague `.json` files
4. Click **Generate Schedule** to see the result
